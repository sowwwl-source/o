Ops notes

Keep deployment plain

Deploy

  • Copy deploy/.env.production.example to deploy/.env.production.
  • Set a real AZA_API_TOKEN and strong secrets.
  • Launch with docker compose --env-file deploy/.env.production -f deploy/docker-compose.prod.yml up --build -d.

DNS

  • Point the hub, API, user cloud, and alternate ingress at the same VPS.
  • Keep www redirects at the proxy layer.
  • Use Cloudflare Full (strict) once origin certificates are healthy.
